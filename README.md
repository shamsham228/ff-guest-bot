# 🔥 Free Fire Guest Bot

Fully free, 24/7 Telegram bot that creates Free Fire guest accounts.
No PC required. Runs on Render.com free tier.

## Features
- `/getguest 10` parallel account creation
- Proxy rotation with auto-ban detection
- JWT token generation
- Web dashboard with live stats
- Auto-proxy scraping from free lists
- 24/7 uptime via Render + UptimeRobot

## Setup
See deployment guide below.

## Commands
- `/getguest [count]` — Create accounts
- `/stats` — View statistics  
- `/proxies` — Proxy status
- `/dashboard` — Dashboard link
